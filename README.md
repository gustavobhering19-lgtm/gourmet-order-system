# 🍨 Sistema de Delivery para Cremosinhos Gourmet

Um sistema de pedidos desenvolvido para uma loja de **Cremosinhos Gourmet**, permitindo que os clientes montem seus pedidos de forma simples e intuitiva.

O projeto foi desenvolvido utilizando apenas **HTML, CSS e JavaScript**, simulando uma experiência completa de compra com cálculo automático, validação de pedido mínimo e estrutura preparada para integração com WhatsApp.

# ✨ Funcionalidades

- 🍓 Seleção de diversos sabores
- ➕ Aumentar ou diminuir quantidades
- 💰 Preço individual por sabor
- 📊 Cálculo automático do valor total
- 📦 Controle da quantidade total
- 🚫 Pedido mínimo de 40 unidades
- 📈 Barra de progresso indicando o pedido mínimo
- 🛒 Carrinho atualizado em tempo real
- ❌ Remoção rápida de itens do carrinho
- 📄 Resumo completo do pedido
- 📋 Botão para copiar o pedido
- 📱 Estrutura pronta para envio via WhatsApp
- ✅ Modal de confirmação do pedido
- 🎨 Interface responsiva e moderna

---

# 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

---

# 💻 Layout

O projeto foi pensado para oferecer uma experiência semelhante a aplicativos modernos de delivery.

Entre os principais diferenciais estão:

- Interface limpa
- Navegação intuitiva
- Atualização instantânea dos valores
- Feedback visual para o usuário
- Organização dos sabores em cards
- Carrinho lateral fixo

---

# 📱 Fluxo do Pedido

O usuário consegue:

1. Escolher os sabores
2. Definir a quantidade desejada
3. Acompanhar o valor total em tempo real
4. Verificar se atingiu o pedido mínimo
5. Finalizar o pedido
6. Visualizar o resumo completo
7. Copiar ou enviar para o WhatsApp
